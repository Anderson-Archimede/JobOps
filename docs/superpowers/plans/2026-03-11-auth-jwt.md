# Auth JWT RS256 & Profils Utilisateurs - Plan d'Implémentation

**Objectif :** Ajouter un système d'authentification production-ready JWT RS256 multi-utilisateurs avec gestion des profils sans altérer la logique métier existante.

**Architecture :** Le backend utilisera une paire de clés asymétriques RS256 pour signer (private) et vérifier (public) les JWT. Les mots de passe seront hachés avec `argon2id`. Un système de double jeton sera mis en place : `accessToken` (15min, mémoire) et `refreshToken` (7 jours, httpOnly cookie). Le backend gérera les routes d'auth, la vérification via middleware et le blacklistage des refresh tokens via Redis lors du logout. Le frontend gérera l'état via un AuthContext, des routes protégées, et un intercepteur Axios pour le renouvellement automatique (Silent Refresh).

**Stack Technique :** Node.js, Express, jsonwebtoken, argon2, Redis, React, React Router, Axios, TailwindCSS.

```typescript
import { generateKeyPairSync } from 'crypto';
import fs from 'fs';
import path from 'path';

export function generateKeys() {
  const { publicKey, privateKey } = generateKeyPairSync('rsa', {
    modulusLength: 2048,
    publicKeyEncoding: {
      type: 'spki',
      format: 'pem',
    },
    privateKeyEncoding: {
      type: 'pkcs8',
      format: 'pem',
    },
  });

  const orchestratorDir = path.join(process.cwd(), 'orchestrator');
  const authKeysDir = path.join(orchestratorDir, 'auth-keys');

  if (!fs.existsSync(authKeysDir)) {
    fs.mkdirSync(authKeysDir, { recursive: true });
  }

  fs.writeFileSync(path.join(authKeysDir, 'private.pem'), privateKey);
  fs.writeFileSync(path.join(authKeysDir, 'public.pem'), publicKey);

  console.log('✅ RS256 keys generated successfully in orchestrator/auth-keys/');
}

if (require.main === module) {
  generateKeys();
}
```

```typescript
import jwt from 'jsonwebtoken';
import fs from 'fs';
import path from 'path';

// Les clés générées dans auth-keys
const privateKey = fs.readFileSync(path.join(process.cwd(), 'auth-keys', 'private.pem'), 'utf8');
const publicKey = fs.readFileSync(path.join(process.cwd(), 'auth-keys', 'public.pem'), 'utf8');

export const signToken = (payload: object, expiresIn: string | number) => {
  return jwt.sign(payload, privateKey, { algorithm: 'RS256', expiresIn });
};

export const verifyToken = (token: string) => {
  try {
    return jwt.verify(token, publicKey, { algorithms: ['RS256'] });
  } catch (error) {
    return null;
  }
};
```

- [ ] **Étape 1 : Implémenter le middleware authenticateJWT**

```typescript
import { Request, Response, NextFunction } from 'express';
import { verifyToken } from '../auth/jwt';

export const authenticateJWT = (req: Request, res: Response, next: NextFunction) => {
  const authHeader = req.headers.authorization;

  if (authHeader && authHeader.startsWith('Bearer ')) {
    const token = authHeader.split(' ')[1];
    const decoded = verifyToken(token);

    if (decoded) {
      // @ts-ignore - attacher l'utilisateur à la requête
      req.user = decoded;
      return next();
    }
  }

  return res.status(401).json({ ok: false, error: { code: 'UNAUTHORIZED', message: 'Token invalide ou manquant' } });
};
```

- [ ] **Étape 1 : Implémenter les routes (register, login, refresh, logout, me)**

```typescript
import { Router } from 'express';
import argon2 from 'argon2';
import { signToken, verifyToken } from '../../auth/jwt';
import { authenticateJWT } from '../../middleware/auth';
import { loginRateLimiter } from '../../middleware/rateLimit';
import Redis from 'ioredis';

const router = Router();
const redisClient = new Redis(process.env.REDIS_URL || 'redis://localhost:6379');

// En mémoire temporaire pour les utilisateurs (À remplacer par la DB Drizzle)
const users: any[] = []; 

router.post('/register', async (req, res) => {
  const { email, password, profile } = req.body;
  const hash = await argon2.hash(password);
  const newUser = { id: Date.now().toString(), email, passwordHash: hash, profile };
  users.push(newUser);
  res.json({ ok: true, data: { id: newUser.id, email, profile } });
});

router.post('/login', loginRateLimiter, async (req, res) => {
  const { email, password } = req.body;
  const user = users.find(u => u.email === email);
  if (!user || !(await argon2.verify(user.passwordHash, password))) {
    return res.status(401).json({ ok: false, error: { code: 'UNAUTHORIZED', message: 'Identifiants incorrects' } });
  }

  const payload = { userId: user.id, email: user.email };
  const accessToken = signToken(payload, '15m');
  const refreshToken = signToken(payload, '7d'); // Le refreshToken peut être stocké côté DB ou Redis

  res.cookie('refreshToken', refreshToken, { httpOnly: true, secure: process.env.NODE_ENV === 'production', sameSite: 'strict', maxAge: 7 * 24 * 60 * 60 * 1000 });
  res.json({ ok: true, data: { accessToken, user: { id: user.id, email: user.email, profile: user.profile } } });
});

router.post('/refresh', async (req, res) => {
  const refreshToken = req.cookies.refreshToken;
  if (!refreshToken) return res.status(401).json({ ok: false, error: { code: 'UNAUTHORIZED', message: 'Refresh token manquant' } });
  
  // Vérifier si blacklisté (Logout)
  const isBlacklisted = await redisClient.get(`bl_${refreshToken}`);
  if (isBlacklisted) return res.status(401).json({ ok: false, error: { code: 'UNAUTHORIZED', message: 'Token révoqué' } });

  const decoded = verifyToken(refreshToken) as any;
  if (!decoded) return res.status(401).json({ ok: false, error: { code: 'UNAUTHORIZED', message: 'Refresh token invalide' } });

  const newAccessToken = signToken({ userId: decoded.userId, email: decoded.email }, '15m');
  res.json({ ok: true, data: { accessToken: newAccessToken } });
});

router.post('/logout', authenticateJWT, async (req, res) => {
  const refreshToken = req.cookies.refreshToken;
  if (refreshToken) {
    const decoded = verifyToken(refreshToken) as any;
    if (decoded) {
      const exp = decoded.exp - Math.floor(Date.now() / 1000);
      if (exp > 0) {
        await redisClient.set(`bl_${refreshToken}`, 'true', 'EX', exp); // Blacklist in Redis
      }
    }
  }
  res.clearCookie('refreshToken');
  res.json({ ok: true, data: { message: 'Déconnecté avec succès' } });
});

router.get('/me', authenticateJWT, (req, res) => {
  // @ts-ignore
  const user = users.find(u => u.id === req.user.userId);
  if (!user) return res.status(404).json({ ok: false, error: { code: 'NOT_FOUND', message: 'Utilisateur non trouvé' } });
  res.json({ ok: true, data: { id: user.id, email: user.email, profile: user.profile } });
});

export default router;
```
